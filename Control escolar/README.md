Sistema de Gestión de Control Escolar

Descripción del proyecto

El presente proyecto consiste en el diseño de un sistema de Control Escolar con fines académicos. Su finalidad es apoyar en la administración de información relacionada con alumnos, docentes, grupos y asignaturas dentro de una institución educativa.

Actualmente, el sistema se encuentra en una fase inicial centrada en el diseño de interfaces (UI/UX) mediante Figma, donde se definieron las principales pantallas y el flujo de navegación.

Componentes del sistema

Módulo de acceso

Permite la autenticación y gestión de usuarios:

* Inicio de sesión (Login)
* Registro de usuarios
* Cierre de sesión (Logout)

Gestión de alumnos

Este módulo permite la administración de los estudiantes:

* Consulta de registros
* Visualización detallada
* Grupo asignado
* Generación y descarga de información en PDF
* Registro de alumnos
* Edición de datos
* Eliminación de registros

Gestión de docentes

Incluye funcionalidades para la administración del personal docente:

* Consulta general
* Visualización detallada
* Exportación de información en PDF
* Registro de docentes
* Modificación de datos
* Eliminación

Gestión de grupos

Permite la organización de alumnos en grupos:

* Consulta de grupos
* Visualización de detalles
* Descarga de información en PDF
* Creación de grupos
* Asignación de alumnos  
* Asignación de docente
* Relación con asignaturas
* Edición
* Eliminación

Gestión de asignaturas

Permite la administración de las materias:

* Consulta de asignaturas
* Visualización de detalles
* Relación con grupos
* Generación de reportes en PDF
* Registro de asignaturas
* Modificación
* Eliminación

Herramientas utilizadas

* Figma (prototipado de interfaces)
* Diseño UX/UI

Forma de uso

1. Acceder al enlace del prototipo en Figma.
2. Navegar entre las pantallas del sistema.
3. Revisar el flujo de interacción.
4. Analizar la estructura de cada módulo.

Estado actual

El proyecto se encuentra en fase de diseño, por lo que no cuenta aún con implementación en código (frontend o backend).
